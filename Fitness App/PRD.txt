📄 PRODUCT REQUIREMENTS DOCUMENT (PRD)
1. Product Overview

Project Name: StrideX (Working Title)
Platform: .NET MAUI (.NET 8)

Vision

StrideX is a modern fitness tracking mobile application built with .NET MAUI that enables users to:

Track physical activities using GPS

Record routes and calculate distance

Monitor daily, weekly, and monthly progress

View past activity history

Discover suggested walking/running locations

Experience a clean modern UI with Light and Dark mode

The application focuses on core tracking functionality without complex social or cloud systems to keep the scope realistic and achievable.

6. Technical Requirements
6.1 APIs (Core System)
🟢 GPS Tracking API

Use:
MAUI Geolocation API
Namespace:

Microsoft.Maui.Devices.Sensors.Geolocation


Purpose:

Continuous location updates

Retrieve Latitude & Longitude

Get Speed

Track accuracy

Alternative:

Android Fused Location Provider (Advanced)

🟢 Map API

Use:
MAUI Maps
Namespace:

Microsoft.Maui.Controls.Maps


Requires:

Google Maps API Key

Alternative:

Mapsui (OpenStreetMap)

🟢 Distance Calculation

No API required.

Algorithm:

Haversine Formula

Distance between coordinate points

Accumulate total distance

🟢 Charts API

Primary:

Microcharts.Maui

Alternative:

LiveCharts2 (More complex)

🟢 Notifications API

Library:

Plugin.LocalNotification

Used for:

Activity reminders

Weekly summaries

🟢 Suggested Locations API

Recommended:

Hardcoded JSON file

Optional:

Google Places API (Nearby Parks Search)
